# Descrição
API REST utilizando Spring Boot.

# Funcionalidades Implementadas

* DTO (com Object Mapper)
* Spring Security
* Swagger (com Springdoc + "Authorize" com Bearer Token no Config)
* Envio de E-mail (com JavaMailSender)
* Tratamento de Exceções
